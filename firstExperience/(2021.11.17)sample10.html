class 선택자 사용하기

-class 선택자를 사용하면 태그 이름이 다른 여러 요소를 찾아 동일한 스타일을 적용할 수 있디.

- 사용하는 방법은 찾고자 하는 클래스 속성 값 앞에 .(마침표)를 붙여 셀렉터를 사용한다.

- 특정 태그에만 해당하는 클래스 값을 지닌 요소를 찾고싶으면 태그명.클래스값으로 찾을 수 있다.

id 선택자 사용하기

-id라는 글로벌 속성을 사용하여 해당 속성값이 일치하는 요소를 찾아 스타일을 적용할 수 있다.
-사용하는 방법은 찾고자하는 id속성 값 앞에 #을 붙여 사용한다.
-특정 태그에 id값이 일치하는 요소를 찾을 때에는 태그명#id속성값 으로 찾을 수 있다.
